Detailed Description

Provides a user interface loop for interacting with WeatherLog operations.

The Menu class acts as the primary interface between the user and the system. It owns a WeatherLog instance and provides a menu-driven loop where users can select operations such as loading data, viewing statistics, and exporting results.
