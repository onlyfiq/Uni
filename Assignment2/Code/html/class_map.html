Detailed Description

template<class K, class V>
class Map< K, V >

A lightweight wrapper around std::map providing simplified access methods.

This templated Map class behaves similarly to std::map but exposes only the features required by the assignment. It supports key lookup, element access, size retrieval, clearing, and extracting all keys into a Vector container.

Template Parameters
KType of the map keys.
VType of the mapped values.
