Detailed Description

Represents a single weather observation.

A WeatherRec object stores:

  • The date of observation
  • The time of observation
  • Wind speed (m/s)
  • Solar radiation (W/m²)
  • Ambient air temperature (°C)

Each record can generate a DateTimeKey, which provides a unique and fully sortable timestamp suitable for use as the key in a BST.
