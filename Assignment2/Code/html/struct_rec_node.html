Detailed Description

Node wrapper storing a WeatherRec and its sortable DateTimeKey for BST usage.

RecNode acts as the element type stored in the binary search tree. Each node contains a WeatherRec and an associated DateTimeKey that determines ordering within the BST. Comparison operators use only the DateTimeKey, ensuring that tree operations are based strictly on chronological order.
