19 *
20 * The WeatherLog class manages all weather data used by the application.
21 * Data is loaded from one or more CSV files whose filenames are listed inside
22 * **data/data_source.txt**.
23 *
24 * Internally, data is stored in a three-level structure:
25 *
26 * Map<year, Map<month, Bst<RecNode>>>

28 * Each RecNode contains a WeatherRec and is inserted into the BST using
29 * a DateTimeKey, ensuring chronological ordering and fast searching.
30 *
31 * The class provides methods for computing:
32 * - Average wind speed (with standard deviation)
33 * - Average ambient air temperature (with standard deviation)
34 * - Total solar radiation
35 * - Pearson correlation coefficients (S/T/R)
