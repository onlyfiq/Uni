\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Bst$<$ T $>$}
A templated Binary Search Tree (BST) class. 

This class provides functionality for storing elements in a binary search tree structure. Supported operations include insertion, searching, deletion, and depth-\/first traversals (in-\/order, pre-\/order, post-\/order).

The type T must support \texorpdfstring{$<$}{<}, \texorpdfstring{$>$}{>}, and == comparison operators.
